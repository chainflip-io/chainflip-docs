# Get Assets

## getAssets

Fetches the list of available assets for the given `chain`.

```typescript copy
getAssets(chain: Chain): Promise<AssetData[]>
```

{<table><thead><tr><th width="223">Param</th><th width="372">Description</th><th width="244.33333333333331">Data type</th></tr></thead><tbody><tr><td><code>chain</code></td><td>The ID of the chain the returned assets</td><td><code>string</code></td></tr></tbody></table>}

### Example

```typescript copy
import { Chains } from "@chainflip-io/chainflip-sdk/swap";

console.log(await swapSDK.getAssets(Chains.Ethereum));
```
