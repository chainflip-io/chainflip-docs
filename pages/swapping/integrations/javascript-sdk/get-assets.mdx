# Get Assets

## getAssets

Fetches the list of available assets for the given `chain`.

```typescript copy
getAssets(chain: Chain): Promise<AssetData[]>
```

### Example

```typescript copy
import { Chains } from "@chainflip/sdk/swap";

console.log(await swapSDK.getAssets(Chains.Ethereum));
```
